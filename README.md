# 🍔 Food Ordering Behavior Dashboard Analysis

## 📌 Project Overview

This project analyzes over **50,000 food ordering records** across India to uncover customer behavior, ordering trends, delivery performance, and business insights.

The analysis was performed using **Microsoft Excel** with **Power Query, Pivot Tables, Pivot Charts, Slicers, and Interactive Dashboard Design** to transform raw data into actionable business insights.

# 🎯 Objectives

- Analyze customer ordering patterns across different cities and time periods.
- Identify the most popular cuisines, meal types, and restaurant categories.
- Evaluate the impact of discounts on customer purchasing behavior.
- Measure customer satisfaction through ratings and repeat orders.
- Assess delivery performance using delivery fees and delivery times.
- Examine how mood, weather, and company influence food ordering behavior.
- Build an interactive dashboard to support data-driven decision-making.

# 🔄 Data Workflow

The project followed these steps:

1. Imported the raw dataset into Excel.
2. Cleaned and transformed the data using Power Query.
3. Checked for missing values and duplicate records.
4. Standardized data formats.
5. Created Pivot Tables for analysis.
6. Built Pivot Charts for visualization.
7. Added interactive slicers for filtering.
8. Designed an interactive dashboard for storytelling.

# 🔍 Analysis & Key Insights

The dashboard revealed several important business insights:

- Premium restaurants generated the highest revenue.
- Weekend orders significantly exceeded weekday orders.
- Rainy weather encouraged higher food ordering activity.
- Repeat customers generally gave higher ratings.
- Discounts positively influenced customer purchasing behavior.
- Faster delivery resulted in higher customer satisfaction.

---

# 💡 Recommendations

Based on the analysis:

- Increase promotional campaigns during weekends.
- Improve delivery efficiency to boost customer satisfaction.
- Expand partnerships with high-performing restaurant categories.
- Introduce personalized offers for repeat customers.
- Optimize discount strategies to maximize profitability.

# ⚠ Assumptions & Limitations

- Dataset represents historical customer behavior only.
- Festivals, inflation, and economic factors were not included.
- Customer locations were limited to selected Indian cities.
