1. 📌 Project Overview

This project analyzes a Food Ordering Behavior Dataset containing 50,000 customer orders across India. The goal was to uncover customer ordering patterns, delivery performance, customer satisfaction, and business insights using Microsoft Excel.

The project demonstrates the complete data analysis workflow, from data cleaning and transformation to interactive dashboard creation for business decision-making.

---

2. 🎯 Objectives

• Analyze customer ordering patterns across different cities and time periods.
• Identify the most popular cuisines, meal types, and restaurant categories.
• Evaluate the impact of discounts on customer purchasing behavior.
• Measure customer satisfaction through ratings and repeat orders.
• Assess delivery performance using delivery fees and delivery times.
• Examine how factors such as mood, weather, and company influence food ordering behavior.
• Build an interactive dashboard that enables stakeholders to explore insights and support data-driven decision-making.

5. 🔄 Data Workflow
The project followed these steps:

• Imported the raw dataset into Excel.
• Cleaned and transformed the data using Power Query.
• Checked for missing values and duplicate records.
• Standardized data formats.
• Created Pivot Tables for analysis.
• Built Pivot Charts for visualization.
• Added interactive slicers for filtering.
• Designed a dashboard to communicate business insights

9. 🔍 Analysis & Key Insights

Some key findings include:

• Premium restaurants generated the highest revenue.
• Weekend orders were significantly higher than weekdays.
• Customers ordering during rainy weather showed increased purchase activity.
• Repeat customers generally provided higher satisfaction ratings.
• Discounts positively influenced customer ordering behavior.
• Delivery performance affected customer ratings.

10. 💡 Recommendations

11. ⚠ Assumptions & Limitations
• Dataset represents historical customer behavior only.
• External factors such as festivals and inflation were not considered.
• Customer locations were limited to selected cities.
